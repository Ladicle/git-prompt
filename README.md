# Git Prompt

Show git status for my fish prompt!
(required Nard-Font)

![screenshot](screenshot.png)

## Installation

Homebrew:

```console
brew install Ladicle/tap/git-prompt
```

Go:

```console
go install github.com/Ladicle/git-prompt@latest
```

See [sample.fish](sample.fish).
